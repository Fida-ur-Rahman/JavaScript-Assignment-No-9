# Print Even Numbers (0-100)

## Overview
This project prints all even numbers from 0 to 100 using a loop. It demonstrates basic iteration and conditional checking in JavaScript.

## Concepts Covered
- Loops (`for` / `while`)  
- Conditional Statements (`if`)  
- Modulus Operator (`%`)  
- Basic Problem Solving  

## Usage
1. Open the JavaScript file in a browser or Node.js environment.  
2. Run the script to display all even numbers from 0 to 100 in the console.
